Instalación en WSL (Ubuntu)

Este asistente proporciona un script descargable para preparar WSL/Ubuntu y lanzar los servicios requeridos (Redis, API y bot).

Paso 1 — Descargar el script

Haz clic para descargar el script de instalación:

Descargar setup_wsl.sh

Paso 2 — Ejecutar en WSL

Abre una terminal WSL (Ubuntu) y ejecuta:

curl -sS --show-error -L "http://localhost:5500/scripts/setup_wsl.sh" -o setup_wsl.sh && chmod +x setup_wsl.sh && sudo ./setup_wsl.sh

Si prefieres copiar el comando manualmente:

Notas

  • El script instala dependencias comunes: Python3, Redis, Docker (opcional) y crea un entorno virtual para `python_api`.
  • Revisa el contenido del script antes de ejecutarlo. Ver script.
  • El servidor web que sirve esta página debe ejecutarse en Windows y exponer el repo (ej. `live-server` o `python -m http.server 5500` desde la carpeta raíz del repo).
