python_bot

Esqueleto inicial de una implementación en Python para DBTeamV2.

Estructura:

- `main.py` - punto de entrada
- `bot.py` - clase Bot y loader de plugins
- `config.py` - configuración mínima
- `plugins/` - plugins Python
- `lang/` - archivos de idioma en Python

Para ejecutar (después de instalar dependencias):

```bash
python -m python_bot.main
```
